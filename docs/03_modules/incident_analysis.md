# Incident Analysis Module

This module performs autonomous investigation of production incidents.

---

## Inputs
- Incident metadata
- Structured logs
- Aggregated metrics
- Trace summaries

---

## Analysis Steps

1. Correlate signals by time and dependency
2. Identify anomaly patterns
3. Build causal incident graph
4. Rank root cause hypotheses
5. Attach historical evidence
6. Assign confidence scores

---

## Key Characteristics

- Deterministic signal correlation
- AI-assisted explanation
- Graph-based causality reasoning
- Read-only analysis

---

## Outputs
- Root cause hypotheses
- Evidence chains
- Incident causal graph
- Confidence scores
